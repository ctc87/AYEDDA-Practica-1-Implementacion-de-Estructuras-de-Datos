vector vector::operator+ ( const vector &  explicito)

Sobrecarga del operador suma.

Lo que hace esta clase al sumar es concatenar al sumando izquierdo el sumando derecho.
Cabe destacar que la concatenación se realiza por la derecha.
ejemplo:
vector[d1][d2][d3]...[dn] + vector[f1][f2][f3]...[dm] = vector[d1][d2][d3]...[dn][f1][f2][f3]...[fm]
Donde d y f son datos y n y m los tamaños de los vectores a sumar.

Se crea un vector auxiliar del tamaño del vector los dos vectores participes en la suma. Después se vuelcan los datos del vector izquierdo en el primer segmento del vector y los del vector derecho en el segundo a través de un bucle for y un if.

Parámetros
explicitosumando derecho;
Devuelve
vector vector que contiene ambos concatenados.

tipoDato & vector::operator[] ( const unsigned  nIndex)

Sobrecarga del operador indexación.

Sobrecargamos el operador []: indexación, para poder acceder al dato apuntado por dicho índice.
ejemplo:
vector[d1][d2][d3]...[dn] -> vector[2] accede a al segundo dato en la memoria reservada dinámicamente. y no a un elemento [2] de un array de objetos vector.
Donde d es un dato y n el tamaño del vector.

Se retorna el dato de la memoria dinámicamente reservado en esa posición.

Parámetros
nIndexNúmero de índice de la posición a la que se quiere acceder;
Devuelve
tipoDato Retorna el dato en la posición indicada.

void vector::reSize ( unsigned  sz)

Este método permite redimensionar el vector a un nuevo tamaño.

Es necesario mencionar que si el tamaño es menor se trunca por la derecha.
ejemplo:
vector[d1][d2][d3]...[dn]-> vector[d1][d2][d3]...[dm]
Donde d es un dato n es el antiguo tamaño y m el nuevo.

El proceso se hace creando un vector auxiliar y copiando dato a dato los contenidos en el vector original. Liberamos la memoria reservada con delete y reservamos memoria de nuevo para el nuevo tamaño por ultimo copiamos posición a posición desde el vector auxiliar a nuestro objeto en la nueva zona de memoria reservada. En caso de no tener datos se inicializa, con new y se le asigna el tamaño sz a numPos.

Parámetros
szNuevo tamaño para el vector.
